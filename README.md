# Celk E2E Testing - Cypress

## Setting up the Environment

* Install Node.js: Visit the official website, download, and install the version suitable for your computer.

* Install VSCode.

* Install plugins in VSCode:

    * Cucumber (Gherkin) Full Support: For Cucumber snippets and templates.
    * vscode-icons: For improved visualization of project file icons.

* Install Cypress: `npm install cypress --save-dev` (Note: antivirus software may block this).

* Install other project dependencies: `npm install`.

## Installing Packages

To install project dependencies:

- open the terminal and navigate to the folder where the project is installed;
- in the same directory as the 'package.json' file, run the command ```npm install```;
- then, complete the installation of files for Cypress, and finally, execute: ```npx cypress open```.
